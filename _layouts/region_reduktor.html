Технические характеристики

Редуктор — это устройство, которое служит для механической передачи крутящего момента коленчатого вала.

Основная характеристика редуктора — передаточное число, это отношение числа зубьев ведомой шестерни к числу зубьев ведущей шестерни. Редукторы КАМАЗ выпускаются со следующими передаточными числами: 4.98, 5.11, 5.43, 5.55, 5.94, 6.33, 6.53, 6.88, 7.22. Чем больше передаточное число, тем больше «тянет» автомобиль: так, для горной местности рекомендуют использовать редуктор с числом 7.22, для равнинной - 5.94, для пересеченной - 6.53. Для автомобилей без прицепов с числом 6.53, 5.94 и 5.43.
